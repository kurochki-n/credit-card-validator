# Credit Card Validator

Консольное приложение на Go для проверки подлинности номеров кредитных карт с использованием алгоритма Луна и идентификации банка-эмитента по BIN.

## Описание

**Credit Card Validator** — это инструмент командной строки, который:

- Проверяет корректность формата номера карты (13-19 цифр по стандарту ISO/IEC 7812)
- Валидирует номер карты с помощью алгоритма Луна (контрольная сумма)
- Определяет банк-эмитент по первым 6 цифрам (BIN — Bank Identification Number)
- Предоставляет интерактивный интерфейс для многократной проверки карт

## Использование

После запуска программа запрашивает номер карты в интерактивном режиме:

```
Введите номер карты> 4000001234567899
Банк: Lunar Bank

Введите номер карты: 5000001234567890
Номер карты недействителен

Введите номер карты>
```

Для выхода из программы нажмите Enter без ввода номера.

### Примеры валидных номеров

| Номер карты        | BIN диапазон    | Банк                   |
|--------------------|-----------------|------------------------|
| 6000001234567891   | 600000-699999   | Venus Express Bank     |

## Алгоритм Луна

Алгоритм Луна (Luhn algorithm) — это простой метод контрольной суммы, используемый для валидации номеров кредитных карт:

1. Начиная с последней цифры (контрольной), двигаемся справа налево
2. Каждую вторую цифру удваиваем
3. Если результат удвоения > 9, вычитаем 9
4. Суммируем все цифры
5. Если сумма кратна 10, номер валиден

## Формат файла banks.txt

Файл содержит данные о банках в формате CSV:

```
НАЗВАНИЕ_БАНКА,BIN_ОТ,BIN_ДО
```

Пример:
```
Lunar Bank,400000,499999
Mars Credit Union,500000,599999
Venus Express Bank,600000,699999
```

Вы можете добавить свои банки, отредактировав файл `banks.txt`.
